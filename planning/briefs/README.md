# Content Briefs

> Detailed content briefs for Lexgro programmatic SEO content

---

## How to Use These Briefs

Each brief contains everything needed to create the content:

1. **Overview** - URL, title, keywords, word count, schema
2. **Search Intent** - Who is searching and what they want
3. **Target Audience** - Demographics and psychographics
4. **Pain Points** - Reddit-sourced frustrations to address
5. **Content Outline** - Full section-by-section structure
6. **Internal Linking** - Exact links to include
7. **Perplexity Prompt** - Copy/paste to research cited sources
8. **Voice Reminders** - Alignment with Keith Dyer's style

---

## Content Creation Workflow

```
1. Read the brief thoroughly
2. Copy Perplexity prompt → Run research → Save sources
3. Draft content following outline
4. Review against Brand Voice Style Guide
5. Add internal links per linking map
6. Implement schema markup
7. Review checklist before publishing
```

## Brief Naming Convention

```
[TYPE]-[NUMBER]-[slug].md

Types:
- PILLAR = Comprehensive guide (5,000+ words)
- ANSWER = Featured snippet page (800-1,500 words)
- SERVICE = Conversion page (2,000-2,500 words)
- BLOG = Thought leadership (1,500-2,500 words)

Examples:
- PILLAR-01-fractional-cmo-law-firms.md
- ANSWER-01-what-is-fractional-cmo.md
- SERVICE-01-fractional-cmo.md
- BLOG-01-why-agencies-fail.md
```
