## Link Velocity Guidelines

### New Page Launch Protocol

When publishing a new page:

1. **Internal links TO new page:**
   - Add link from pillar page (if answer/blog)
   - Add link from 2-3 related answer pages
   - Add link from 1 related blog post
   - Update any existing content that references the topic

2. **Internal links FROM new page:**
   - Link to pillar page (required)
   - Link to 2-3 related answers
   - Link to service page (CTA)
   - Link to 1 related blog post

### Anchor Text Distribution

| Type | Percentage | Example |
|------|------------|---------|
| Exact match | 20% | "fractional CMO for law firms" |
| Partial match | 40% | "fractional CMO services", "law firm fractional CMO" |
