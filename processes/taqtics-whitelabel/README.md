# Taqtics White-Label Processes

Standardized processes and templates for LEXGRO client deliverables.

## Available Processes

### Backlink Reporting
`/backlink-reporting/`

Comprehensive backlink analysis and link building strategy reports.
- 10-page PDF template
- Link gap analysis
- Profile link audits
- Outreach templates
- 90-day action plans

**Example:** MesoWatch.org report (January 2026)

---

## Adding New Processes

1. Create folder under `/processes/taqtics-whitelabel/`
2. Include:
   - `README.md` with process overview
   - Template files (Python scripts, document templates)
   - `/examples/` folder with working client examples
3. Document all client data requirements
4. Include brand customization options

---

## Standards

- All reports branded as **LEXGRO (LexxlyRank)**
- Client data stored at `client-domain.com/data/`
- PDF output for client delivery
- Raw data exports in JSON/CSV
